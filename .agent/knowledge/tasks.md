# Tasks Knowledge

This file documents the task‑runner domain for the Ligan project.

## Scope
- Goblin Tools CLI utilities.
- Automation scripts for builds, linting, and testing.
- Task scheduling and background workers.
- Integration with npm scripts and custom PowerShell helpers.

## Key Files
- `src/lib/goblin.ts` – core task library.
- `scripts/*.ps1` – PowerShell helpers for CI/CD.
- `package.json` – npm task definitions.
